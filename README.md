# Wolt: Assignment

### Installation and running

Clone the repo.

```sh
$ git clone https://github.com/xlakostax/wolt-test-assignment.git
```

Install the dependencies and devDependencies in root, and start the server.

```sh
$ cd wolt-test-assignmen
$ npm install
$ npm start
```
